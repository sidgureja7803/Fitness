# AI Fitness Coach

A clean, professional frontend for an AI Fitness Coach web application built with Next.js, Tailwind CSS, and shadcn/ui.

## Features

- **Home Page**: Hero section with app description and "Get Started" CTA
- **Fitness Input Form**: Comprehensive form to collect user fitness details
- **Results Page**: Displays personalized workout and diet plans with mock data
- **About Page**: Information about the application and technologies used
- **Contact Page**: Contact form with office information
- **Dark/Light Theme**: Toggle between themes with persistent preferences
- **Responsive Design**: Optimized for desktop and mobile devices
- **Modern UI**: Built with shadcn/ui components and Tailwind CSS

## Getting Started

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Tech Stack

- **Next.js 13**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **shadcn/ui**: High-quality React components
- **next-themes**: Theme management for dark/light mode
- **Lucide React**: Icon library

## Pages

- `/` - Home page with hero section
- `/fitness-input` - Fitness details input form
- `/results` - Workout and diet plan display
- `/about` - About the application
- `/contact` - Contact form

## Design Features

- Neutral color palette (gray, zinc) for professional appearance
- Inter font family for modern typography
- Smooth transitions and hover effects
- Accessible UI components
- Corporate-style minimal design
- No gradients or bright colors
